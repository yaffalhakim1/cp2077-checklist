# Cyberpunk 2077 — Quest Checklist

An interactive quest tracker for Cyberpunk 2077 and its Phantom Liberty DLC. Keep track of your progress across 150+ quests with persistent save state, smart filtering, and a cyberpunk-themed UI.

## Features

- **150+ Quests Tracked** — Full base game and Phantom Liberty DLC coverage
- **Real-time Progress Bar** — Visual completion percentage updated instantly
- **Smart Filtering** — Filter by All / Main Only / Side Only / Key for Endings / Undone / Completed
- **Stats Dashboard** — Category breakdown for Main Story, Side Missions, and Dogtown/PL quests
- **Persistent Storage** — Progress is saved to browser `localStorage` and survives page refreshes
- **Quest Notes** — Inline tips for missable quests, romance options, and ending requirements
- **Reset Functionality** — Clear all progress and start fresh

## Usage

No installation or build step required. Simply open `index.html` in any modern web browser.

```bash
# Clone the repository
git clone https://github.com/yaffalhakim1/cp2077-checklist.git

# Open directly in your browser
open index.html
```

Or visit the live version hosted via GitHub Pages.

## Tech Stack

- **HTML5** — Single-file application
- **CSS3** — Embedded styles with Cyberpunk 2077-themed design
- **Vanilla JavaScript** — No frameworks or dependencies
- **localStorage API** — Client-side data persistence
